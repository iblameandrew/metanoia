# metanoia

En mi primer libro "Tekton" abordando la feminidad y la masculinidad, adopté la disposición de un profeta del viejo testamento: la comisión del profeta fue reunir de nuevo hombres y mujeres - asi el profeta descubrio la tecnica del principio generativo: no obstante como a Juan el Bautista, al profeta le faltaban muchas cosas. En este nuevo libro, se explora lo que Jesus fue a Juan El Bautista y se redime la enseñanza del profeta dando el advenimiento del super hombre en la tierra. Este libro esta en proceso activo de edición por lo que incluye muchos borradores. Lo hice "open source" por que francamente en esta era, la autoria intelectual ya no vale nada.

La tematica de este libro es densa y dificil; el primer borrador era una parodía y el tono de las ultimas actualizaciones, lo cambiaba a algo que se leia como las Epistolas. Este nuevo repositirio se decide por un solo estilo.
